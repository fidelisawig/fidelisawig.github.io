Project Overview

Wilayah Studi is a sophisticated web application built with React and TypeScript that enables cartographers and GIS professionals to design publication-ready maps with unprecedented ease. The application features a responsive SVG canvas with support for multiple projections.

Key Features

Projection Support: Toggle between Mercator and Equirectangular projections to visualize data in different coordinate systems and assess geographic distortion.

Dynamic Styling: Customize colors, fonts, and spatial layers with real-time preview capabilities.

Export Options: Generate high-resolution SVG exports suitable for publication in academic papers, reports, and professional documentation.

GIS Integration: Import and process vector data from GeoJSON, KML, and GeoPackage formats.

Technical Implementation

The project leverages modern web technologies including Vite for rapid development, TypeScript for type safety, and custom projection mathematics for accurate coordinate transformations. The application includes sophisticated data parsing for multiple geospatial formats and a custom SVG rendering engine.
